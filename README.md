# ERMS

Short description:
- Project: Egypt Railway Management System (ERMS)
- Stack: MERN (Node/Express, MongoDB, React)

## Structure
- backend/: server (Express, Mongoose)
- frontend/: client (React)
- docs/: documentation
- infra/: docker / infra configs

## Quick start
1. Backend:
   cd backend
   npm install
   # set MONGO_URI in .env
   npm run dev

2. Frontend:
   cd frontend
   npm install
   npm run dev
